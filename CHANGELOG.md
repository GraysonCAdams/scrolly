## [1.0.0] - 2026-02-27

### Added
- Video feed with SMS-based clip sharing via Twilio
- Music link sharing with cross-platform resolution via Odesli
- Group management with invite codes and customizable accent colors
- Push notifications via Web Push API
- Favorites, reactions, and comment threads
- Theme system (dark/light/system) per user
- Content retention and storage management
- Docker support for self-hosting
- Health check endpoint at `/api/health`
- Auto-migrations on startup via Drizzle ORM
